## Full Description (max 4000 chars)

Welcome to the personal portfolio app of Murat Keskin!

This app is a modern, clean showcase of my work as a mobile and full-stack
developer — projects, skills, and contact information, all in one place.

FEATURES

• Home — A short bio, profile info, and highlighted stats
• Projects — Project cards with descriptions, tech stack tags, and
  GitHub / live demo links
• Skills — Frontend, Backend, and Tools categories with visual proficiency
  indicators
• Contact — Email and social links (GitHub, LinkedIn, Twitter/X)

The app features a modern dark theme with smooth navigation between screens.
It works fully offline — internet access is only used when you tap an
external link (GitHub, LinkedIn, etc.).

This app contains no ads and does not collect any personal data or require
account creation.
